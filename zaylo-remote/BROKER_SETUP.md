# Zaylo Preview Notes

This folder is the public portfolio demo build. It is intentionally self-contained:

- Firebase authentication is replaced by a local demo user.
- MQTT is simulated in the browser.
- Devices, scenes, automations, and telemetry are seeded by `preview-data.js`.
- No production broker host, broker password, Firebase project, or weather API key is included.

The production app and firmware must use private deployment configuration outside this public demo folder.
